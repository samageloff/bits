Introduction

Bits are basic, configurable web components, made with JavaScript. No external library dependencies.

Doesn't the web have enough of these?

Yes. The web has plenty of these. This is an exercise in simple, dependency-free, web component development. Modules are be initialized using a data-mod attribute, and can be passed an optional configuration object.
